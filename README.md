# 📊 Dashboard de Gestão de Projetos e Eficiência Operacional

## 🎯 Objetivo do Projeto
Este projeto consiste em um dashboard interativo desenvolvido em **Power BI** para monitorar e gerenciar o ciclo de vida de projetos. O objetivo principal é fornecer uma visão clara sobre o avanço físico e financeiro das iniciativas, identificando gargalos operacionais e garantindo que as entregas ocorram dentro do prazo e do orçamento.

Este portfólio reflete a aplicação de **análise de dados (Data Analytics)** na otimização de rotinas de Back Office e Planejamento, facilitando a tomada de decisão baseada em evidências (*Data-Driven*).

## 📖 Contexto e Processo de Desenvolvimento
Este projeto foi desenvolvido como parte de uma jornada prática de aprofundamento em **Data Analytics e Business Intelligence**. Para construir este dashboard, utilizei como base técnica um projeto guiado passo a passo através [desta playlist de tutoriais no YouTube](https://www.youtube.com/playlist?list=PL4yDzx-iZXHqRaf1RFZ8TxEvyXTzqtFpc), o que me permitiu consolidar na prática os fundamentos do Power BI.

Mais do que replicar um visual, o foco deste estudo foi dominar a mecânica "por trás" do painel e entender como essas métricas se aplicam à realidade de rotinas operacionais e de retaguarda corporativa (Back Office). 

Durante a construção deste projeto, desenvolvi e apliquei as seguintes competências técnicas:
*   **Extração e Tratamento (ETL):** Utilização do Power Query para limpar bases de dados brutas, padronizar formatos de data e criar colunas condicionais essenciais para a análise.
*   **Modelagem de Dados:** Estruturação dos relacionamentos entre tabelas (fatos e dimensões) para garantir que os filtros do dashboard conversem entre si sem quebrar os visuais.
*   **Linguagem DAX:** Criação de medidas calculadas para extrair inteligência dos dados, como variação de orçamento, contagem de tarefas críticas e acompanhamento percentual de conclusão.
*   **Visualização e Storytelling:** Aplicação de boas práticas de design de dashboards, utilizando visuais customizados (como o Gantt Chart) para facilitar a leitura de cronogramas complexos por parte de gestores e *stakeholders*.

## 💼 O Problema e a Visão de Negócios
Embora a base técnica tenha sido guiada, a leitura destes dados conecta-se diretamente com minha experiência em **Operações Financeiras e Ciclo de Crédito**. 

Em operações corporativas complexas, a falta de visibilidade sobre o status real dos projetos gera atrasos, estouros de orçamento, ineficiência operacional e má alocação de recursos. Este dashboard reflete a mentalidade *Data-Driven* que busco aplicar em áreas de planejamento e controle para resolver esses problemas, centralizando:
- O acompanhamento de cronogramas e entregáveis.
- A visualização de atrasos e pendências críticas, mapeando gargalos rapidamente.
- A gestão do esforço e carga de trabalho das equipes.
- O fornecimento de evidências claras para que a tomada de decisão seja ágil e segura.

## 📈 Principais Funcionalidades e Visuais
- **Acompanhamento de Cronograma:** Utilização de Gráfico de Gantt avançado para visualização de fases, dependências e marcos (Milestones).
- **Indicadores de Desempenho (KPIs):** Monitoramento de taxa de conclusão (%), orçamento previsto vs. realizado e status de saúde do projeto.
- **Filtros Dinâmicos:** Segmentação de dados por gerente de projeto, departamento, status (Em andamento, Concluído, Atrasado) e período.
- **Identificação de Gargalos:** Tabelas detalhadas destacando tarefas críticas que necessitam de intervenção imediata.

## 🛠️ Ferramentas e Tecnologias Utilizadas
- **Power BI:** Desenvolvimento de visuais, painéis interativos e modelagem de dados.
- **Power Query:** Limpeza, transformação e estruturação da base de dados (ETL).
- **DAX (Data Analysis Expressions):** Criação de medidas calculadas.
- **Gráficos Customizados:** Implementação do *Gantt Chart by MAQ Software* para cronogramas.

## 🚀 Como visualizar o projeto
1. Baixe o arquivo `Gerenciamento de projetos 2.1.pbix` disponível neste repositório.
2. É necessário ter o [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) instalado em sua máquina.
3. Abra o arquivo com o Power BI Desktop para interagir com os filtros e analisar as medidas DAX construídas.
